- **红色** 的框是动画方法所属的组件类，例如 `Transform` 、 `Image` 等。
- **蓝色** 的框是方法的归类，就是绿色框的归类，移动方法在这里、旋转方法在这里。
- **绿色** 的框是红色组件 `Transform` 所有的动画方法，用起来就是 `Transform.DOMove()`
